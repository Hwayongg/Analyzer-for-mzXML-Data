# 목표 : .mzXML형식의 데이터중 필요한 값들을 추출하여 2가지 그래프를 플로팅
# 목표 설명 :
	1. 현재 학부 연구생으로 배우고 있는 실험실에서 화합물의 '질량분석'을 연구.
	2. 질량분석 데이터의 경우 질량분석기의 제조사(vendor)가 판매하는 프로그램을 이용해야함.
	3. 이 프로그램의 가격이 비싸 파이썬을 이용한 self 분석소프트웨어개발 연구가 진행되고있음.
	4. .mzXML (앞의 mz는 mass-spectrometry, 질량분석을 의미)의 경우
	    데이터를 처리할 수 있는 함수가 공개되어있음. (github에 있는 함수)
	5. 데이터를 추출해 질량분석에서 가장 기본이 되는 Base Peak Chromatogram과
	    Mass Spectrum의 두 그래프를 플로팅해보자.

# 여기서 mzXML파일 형식이란? :
	1. 질량분석은 긴 시간(1시간 내외)동안 0.0X초 단위로 데이터를 측정
	2. 각 시간에서의 데이터를 scan'N'으로 저장(scan 1, scan2, scan3 ...)
	    총 scan의 수는 X0000개 (3만개 내외)
	3. 이 scan속에는 해당 시간에서 얻은 분자 조각들의 질량분석 값이 저장되어있음.

# 과정 :
	* 최초의 분석 결과인 .raw파일을  -> .mzxml파일 형식으로 변환
	- 파이썬 프로그램 밖에서 공개된 무료 소프트웨어 사용 (파이썬 사용X)

	1. .mzXML파일을 분석 할 수 있는 라이브러리를 import
	2. .mzXML을 파이썬에 로드
	3. 라이브러리를 이용, 필요한 값을 추출하고 적당한 자료형으로 저장
	    * 데이터의 양이 너무 방대하므로 교수님께서 말씀해주신 numpy를 이용한는게 어떨까? 
	- Base Peak Chromatogram : 전체 시간에서
		X축 = retention time (해당 scan의 시간)
		Y축 = base peak intensity (해당 scan 속에 있는 데이터)
	- Mass Spectrum : 특정한 하나의 시간(scan)에서
		X축 = m/z (질량을 전하로 나눈 값, 특정 scan속에 있는 데이터)
		Y축 = intensity (특정 scan속에 있는 데이터)
	4. 이를 적당한 함수를 이용해 그래프로 플로팅
	5. 플로팅 된 그래프를 저장할 수 있도록 프로그래밍


# Project의 구체화

-입력데이터: mzXML형식의 database (sample.mzXML을 준비)
-출력데이터: matplotlib을 이용한 BPS, MS 그래프의 플로팅 및 이미지파일로의 저장
-실행형태: mzXML파일의 path를 사람이 입력 >> 소프트웨어가 자동으로 데이터추출 및 그래프 플로팅, 이미지파일로 저장
